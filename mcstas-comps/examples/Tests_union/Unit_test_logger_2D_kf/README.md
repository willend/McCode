# The `Unit_test_logger_2D_kf` Instrument

*McStas: Test of Union logger 2D kf that records the final wavevector*

## Identification

- **Site:** Tests_union
- **Author:** Mads Bertelsen
- **Origin:** ESS
- **Date:** June 2020

## Description

```text
Test instrument for the Union logger component 2D kf that records the final
wavevector after each scattering event, projects it onto a specified 2D plane
and histograms these using the scattered intensity.

Includes conditional component to test record_to_temp functionality
of investigated logger.
```
