# The `Unit_test_logger_2DQ` Instrument

*McStas: Test of the Union 2DQ logger that shows scattering vector projected onto 2D plane*

## Identification

- **Site:** Tests_union
- **Author:** Mads Bertelsen
- **Origin:** ESS
- **Date:** June 2020

## Description

```text
This instrument uses a basic Union system and investigates scattering within
using the 2DQ logger that histograms occurances of the scattering vectors for
scattering events in the Union system projected onto a chosen 2D plane.

Includes conditional component to test record_to_temp functionality
of investigated logger.
```
